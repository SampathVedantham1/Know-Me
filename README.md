# Know-Me: Personal Profile Application
### Introduction:
- Know-Me is a personalized profile application developed using Jetpack Compose, aimed at providing an interactive and dynamic presentation of personal details, including experience, education, projects, and links to code repositories.

### Features:
- Jetpack Compose UI: The application leverages Jetpack Compose to create a modern and visually appealing user interface, ensuring an engaging user experience.

### Android Tests: 
- Comprehensive Android tests have been integrated to ensure the robustness and reliability of the application.

### Navigation Drawer: 
- A convenient navigation drawer has been implemented, offering quick access to essential features such as a QR code for resume details, a download link to view the resume, and contact details such as phone number and mail id.

### Sections:
- #### Experience:
    - Explore the professional journey with detailed information about work experience, roles, and accomplishments.

- #### Education:
    - Discover academic achievements and educational background, providing insights into the learning journey.

- #### Projects:
    - Gain insights into various projects undertaken, including descriptions, technologies used, and references to code repositories for further exploration.

- #### Navigation Drawer:
    - Scanner: Utilize the QR code to easily access/share resume details.

    - Download Resume: A quick link to download the resume for a comprehensive overview.

### Contact Details: 
- Find ways to get in touch for collaboration or inquiries.

### How to Run :
- ### First method :
    - Navigate to the settings on your mobile device.
    - Grant permission for the installation of the application from Drive by enabling it from "Install Unknown Apps"  option.
    - Proceed to click the "APK" link available in the References section of the README file.
    - Scan the application and initiate the installation process.

 - ### Second method :
    - Clone the repository.
       
    `git clone https://github.com/YourUsername/Know-Me.git`

    - Open the project in Android Studio.
    - Run the application on an emulator or physical device.
